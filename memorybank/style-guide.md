# Style Guide

## Brand Colors
- **Primary (Button/Title)**: #4630d2 (Deep purple/violet)
- **Button Text**: #ffffff (White)
- **Banner/Background**: #e1f5fe (Light blue)
- **Banner Text**: #000000 (Black)
- **Logo**: Purple variant matching primary color

## Typography Hierarchy
- **Headers**: Larger font weights, primary color
- **Subheaders**: Medium weight, secondary styling
- **Body Text**: Standard weight, good contrast
- **Small Text**: Reduced size, muted colors for secondary info

## Design Principles
1. **Consistency**: Use brand colors throughout
2. **Hierarchy**: Clear visual structure with size and color
3. **Readability**: High contrast, appropriate spacing
4. **Professionalism**: Clean, modern aesthetic
5. **Mobile-First**: Responsive design for all devices

## Component Styling Guidelines
- **Buttons**: Primary color background, white text, rounded corners
- **Cards**: Subtle backgrounds, proper spacing, clear borders
- **Headers**: Bold typography, brand colors, good spacing
- **Tables**: Clean rows, alternating colors, clear headers
- **Banners**: Light background, good contrast, prominent info

## Spacing System
- **Tight**: 4-8px for related elements
- **Normal**: 16px for standard spacing
- **Loose**: 24-32px for section separation
- **Extra Loose**: 48px+ for major sections

## Border and Radius
- **Buttons**: Rounded corners (8px radius)
- **Cards**: Subtle rounded corners (4-8px)
- **Containers**: Soft edges for modern look

## Responsive Considerations
- **Mobile**: Stacked layouts, larger touch targets
- **Tablet**: Balanced spacing, readable text
- **Desktop**: Full width layouts, optimal information density
